# prepend * to every non-blank line

Prepend an asterisk to every non-blank line in the input file.

**Start file**

```
This is a
very short

file, but it is 
still
full

of

surpises.
```

**End file**

```
*This is a
*very short

*file, but it is 
*still
*full

*of

*surpises.
```
